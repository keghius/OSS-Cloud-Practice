# ☁️ 오픈소스SW분석 (클라우드) 실습 저장소

**2026년도 1학기 단국대학교 SW융합대학 컴퓨터공학과**

이 저장소는 **오픈소스SW분석(클라우드)** 과목(담당: 남재현 교수님)을 수강하며 진행하는 주차별 실습 코드와 학습 내용을 정리하고 기록하기 위한 공간입니다.

## 📖 교과목 개요

컨테이너 기술의 핵심인 **도커(Docker)**와 컨테이너 오케스트레이션 플랫폼인 **쿠버네티스(Kubernetes)**의 원리를 학습합니다. 컨테이너 기반 애플리케이션의 설계, 배포, 관리 기술을 직접 실습하며, 현대 소프트웨어 개발 및 운영 환경에서 필수적인 **DevOps 및 클라우드 네이티브 컴퓨팅**의 기본 실무 역량을 기르는 것을 목표로 합니다.

* **주교재:** 한 권으로 배우는 도커 & 쿠버네티스 (장철원 저, 한빛미디어)
* **수업방식:** PBL(문제해결학습) 기반 이론 및 실습 병행

## 💻 실습 환경

* **OS:** macOS
* **도구:** Docker Desktop, Kubernetes
